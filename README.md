# Agent GoG Use Policy (Sample)

A lightweight, sample policy document that describes guardrails for agents using the `gog` CLI to interact with Google Drive.

## Why this exists
- Provide a clear, human-readable policy for what an agent **may** and **may not** do.
- Encourage least-privilege access and explicit approvals for destructive actions.
- Offer a template your org can adapt to its own Drive structure and governance needs.

## Contents
- `agent-gog-use-policy.md` — the sample policy template.

## Usage
- Copy and customize the policy for your environment.
- Replace placeholder IDs with real Drive/Folder IDs.
- Use as a discussion starter for governance and automation best practices.
